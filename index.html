    • Displacement with THREE.js

    This animated displacement effect was created using THREE.js. Specifically, post-proccesing was used to alter the original image, combining glsl shaders with vanilla Javascript. The simpler animations were done with GSAP.
    Three.js is a cross-browser JavaScript library and application programming interface (API) used to create and display animated 3D computer graphics in a web browser using WebGL. The source code is hosted in a repository on GitHub. GSAP is a set of small JavaScript files that help you create animations that look great and all major browsers jiseop eliminates the headaches caused by browser bugs and quirks.
